# 默认功能输入模板

仅用这个模板收集缺失的功能开发输入。它不是项目规则来源。

## 必填

- 功能目标：`<用户希望新增或修改什么>`
- 核心工作区：`<主要模块或路径，如已知>`
- 验收行为：`<可观察的行为或预期结果>`

## 可选

- 参考区：`<辅助模块、实现、API 或文档>`
- 相关文档：`<路径或 URL>`
- 约束：`<兼容性、依赖、API、编码、性能或发布约束>`
- 验证方式：`<建议的编译、测试或人工检查>`
- 未确认问题：`<编码前需要澄清的问题>`

## 规则

- 空字段不是需求。
- 用户本次提供的值优先于此模板。
- 项目批准模板优先于此模板。
- 源码和项目本地 `docs/ai-coding/` 仍然是权威来源。
